# react-library-test

## Install

```bash
npm install --save react-library-test
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-library-test'
import 'react-library-test/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
